# Governance Agent — 全栈治理与可观测性

## 职责
- **身份认证**：OAuth2 / OIDC，对接 AD / LDAP
- **权限模型**：RBAC + ABAC（知识库/文档/段落级）
- **可观测底座**：Prometheus + Grafana + Loki
- **全链路追踪**：OpenTelemetry（所有模块）
- **LLM 追踪**：Langfuse 自托管（⚠️ 内存紧张可延后）
- **审计日志**：统一格式 + 合规报告导出（PDF/Excel）
- **成本控制**：按项目/部门统计 API 费用 + 预算告警
- **防呆机制**：全平台操作前安全校验（面向非技术用户）

## 防呆机制（Anti-Foolproof）
所有写入/删除操作必须经过防呆校验中间件：
- 检查操作是否可逆
- 警告不可逆操作
- 评估影响范围
- 用通俗语言解释风险
- 非技术用户二次确认
