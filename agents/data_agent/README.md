# Data Agent — 数据情报服务

## 职责
- 多源爬虫框架（YAML 规则配置化）
- 全球数据源采集（新闻/社交媒体/行业论坛/政府数据，中/英/日/韩）
- 数据清洗管道（去重 → 实体识别 → 分类打标 → 结构化）
- 自动分析流水线（摘要/情感分析/趋势识别/关键事件提取）
- 报告模板引擎（支持自定义模板）
- 定制化推送服务（Web 控制台配置 → 生成 → 推送）
