# FDE AI 平台 · 企业信息化培训手册

> 配套 V5「企业落地七步法」③ 培训与认证模块。本手册用于企业信息化培训班授课与考证，覆盖初级 / 中级 / 高级三级认证体系。

---

## 0. 平台定位

FDE AI 平台是面向企业的「AI 落地操作系统」，通过七步法把大模型能力转化为可运营的业务模块：

| 步骤 | 模块 | 一句话价值 |
|------|------|-----------|
| ① 基础 | 信息化基建 | 知识入库 + 统一检索底座 |
| ② 交付 | 定制驾驶舱 | 把数据变成老板看得懂的看板 |
| ③ 培训 | 本模块 | 让人会用、敢用、用出 ROI |
| ④ 情报 | 外部情报增幅器 | 实时监控竞品与行业动态 |
| ⑤ 营销 | GEO 投放 | 让 AI 搜索引擎主动推荐你 |
| ⑥ 裁员 | 智能裁员评估 | 用数据辅助组织优化（含防呆） |
| ⑦ 定价 | 动态定价引擎 | 用弹性与竞品数据自动定价 |

统一入口：`/hub/`（七步法导航页）。

---

## 1. 初级认证（操作员 · Operator）

**目标**：能在指导下完成日常操作，不踩红线。

### 1.1 知识入库
- 登录 `/portal/`，进入「上传」页，支持 xlsx / csv / pdf / docx / pptx。
- 文件经解析 → 归一化 → 分块后自动进入向量库，可用自然语言问答。
- 口诀：**先结构化、再入库、后问答**。

### 1.2 驾驶舱查看
- `/portal/dashboard` 查看企业核心指标看板。
- 看板指标由数据接入层实时刷新，异常会自动标红。

### 1.3 情报订阅
- `/intel/` 在「数据源」中添加 RSS / HTTP / API 源。
- 在「预警」中设置关键词（如竞品名、监管词），命中后生成触发记录。

### 1.4 红线清单（必考）
1. 不得上传含个人敏感信息（身份证号、银行卡）的文件。
2. 不得绕过防呆弹窗直接执行裁员方案。
3. 定价建议仅作决策参考，最终价格须由人审批。

## 2. 中级认证（分析师 · Analyst）

**目标**：能独立配置模块、解读结果、给出业务建议。

### 2.1 GEO 营销
- `/marketing/` 四页：GEO 看板、内容工作室、广告管理、ROI 看板。
- 重点理解 **GEO 指数**（品牌被 AI 搜索引擎引用率）与 **ROAS**（广告支出回报）。
- 实操：用「内容工作室」生成一篇 E-E-A-T 优化的文章，再用「广告管理」做 A/B 变体并看显著性。

### 2.2 动态定价
- `/pricing/` 四页：总览、What-if 模拟、策略优化、弹性分析、竞品监控。
- 理解 **价格弹性**（涨价 1% 销量降多少）与 **需求预测**逻辑。
- 实操：在「弹性分析」读某产品的弹性系数，在「模拟器」里试算不同涨价幅度下的利润。

### 2.3 情报深度使用
- 「趋势」页看 14 日采集曲线与关键词云。
- 「报告」页一键导出 Markdown 简报，按情感过滤噪声。

## 3. 高级认证（架构师 · Architect）

**目标**：能设计落地方案、配置 Dify 工作流、把控合规风险。

### 3.1 Dify 工作流编排
- 导入 `docs/fde-dify-openapi.yaml` 作为 Custom Tool（provider：`fde_data_tool`）。
- 导入 `docs/dify-workflows/` 下的预置工作流：合同分析 / 数据问答 / 情报简报 / 定价建议。
- 可组合 FDE 工具节点与 LLM 节点，搭建行业专用流程。

### 3.2 裁员评估（含防呆五步）
- `/hr/simulator` 选择部门 → 看冗余率 / 年节省 / 风险等级。
- **防呆五步**（系统强制）：① 可逆性检查 → ② 影响范围评估 → ③ 通俗解释 → ④ 二次确认（输「确认裁员」）→ ⑤ 方案快照存档。
- 高级认证须能向管理层解释每一步的合规依据。

### 3.3 合规与审计
- 所有关键操作写入审计日志，可追溯。
- 认证考试含情景题：如何在「降本」与「合规 / 舆情风险」间取舍。

## 4. 培训形式建议

| 形式 | 内容 | 时长 |
|------|------|------|
| 线上视频课 | 各模块操作演示 | 每模块 15 min |
| 直播带练 | 真实数据集演练 | 2h / 场 |
| 沙盘演练 | 模拟裁员 / 定价决策 | 半天 |
| 考证 | 理论 + 实操 + 情景 | 初级 60min / 中级 90min / 高级 120min |

> 注：本手册为文字版大纲，视频课程与题库见本目录 `exam-bank.md` 及后续补充材料。
