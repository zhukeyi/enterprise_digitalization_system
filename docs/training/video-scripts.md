## Production Notes

### Recording Setup
- **Resolution**: 1920x1080 (1080p)
- **Frame rate**: 30 fps
- **Audio**: External microphone, noise-filtered
- **Browser**: Chrome/Edge in incognito mode (clean state)
- **Font size**: Increase browser zoom to 125% for readability

### Editing Guidelines
- Add FDE logo watermark in bottom-right corner
- Use fade transitions between sections
- Add chapter markers at each step
- Highlight mouse clicks with a subtle ripple effect
- Include subtitles in both Chinese and English

### Distribution Format
- Output: MP4 (H.264 + AAC)
- Bitrate: 5 Mbps video, 128 kbps audio
- File naming: `fde-training-01-overview.mp4`, `fde-training-02-ingestion.mp4`, etc.
- Host on platform's `/training/` portal or external LMS
